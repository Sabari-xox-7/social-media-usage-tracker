# 📱 Social Media Usage Tracker & Limiter

A beginner Python project for 1st Year Engineering students.

## 🎯 Features
- Track daily usage of Instagram, YouTube, WhatsApp, etc.
- Set custom daily time limits per app
- Get warnings at 80% and 100% of limit
- Daily and weekly usage reports
- Data saved to JSON file

## 🛠 How to Run
```bash
python social_media_tracker.py
```
